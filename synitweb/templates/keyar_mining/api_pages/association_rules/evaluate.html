Association rule learning is a rule-based machine learning method for discovering interesting relations between variables in large databases. It is intended to identify strong rules discovered in databases using some measures of interestingness. This function is to evaluate the algoritms that got supports

                from krmining.association_rules._fpgrowth import fpgrowth
                from krmining.association_rules._evaluate import evaluate

                from krmining.datasets import make_fruit_sold_dummy_association_rules
                

                df = make_fruit_sold_dummy_association_rules()

                apriori_evaluate = evaluate(combinations_supports, return_df=True)
