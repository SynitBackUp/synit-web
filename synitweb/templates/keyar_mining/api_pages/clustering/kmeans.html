K-Means Algorithm

k-means clustering is a method of vector quantization, originally from signal processing, that aims to partition n observations into k clusters in which each observation belongs to the cluster with the nearest mean (cluster centers or cluster centroid), serving as a prototype of the cluster. This results in a partitioning of the data space into Voronoi cells. k-means clustering minimizes within-cluster variances (squared Euclidean distances), but not regular Euclidean distances, which would be the more difficult Weber problem: the mean optimizes squared errors, whereas only the geometric median minimizes Euclidean distances.

Methods Description
fit(data) training or fitting the model
predict(data) predict where the data belongs
evaluate(data) result evaluating model
Attributes Description
k how much k is used
n_iters how much iteration is used
init what initialize method is use
centroid where is the centroid
Parameters Description
k type : int
desc : number of cluters
n_iters type : int
optional : int number greater than 0
desc : the iteration of k-means algorithm. The default is 10.
init type : string
optional : {"random", "kmeans++"}
desc : the algorithm to determine first centroids. The default is "kmeans++".
data type : numpy.ndarray or shape(n_samples, n_features)
desc : what data to be used

How To

For trying to use the code you can follow the full code in the below

                from krmining.clustering import KMeans
                from krmining.datasets import get_example_covid_id

                df = get_example_covid_id()

                k = 2

                kmeans = KMeans(k)
                kmeans.fit(df)
                centroid = kmeans.centroid
