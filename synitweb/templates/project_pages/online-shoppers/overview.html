Overview Online Shoppers Project

Online Shoppers Project is about project to classify which customer would buy the product or don't buy the product based on the features in the dataset. In this project, i compared each model to classify which buy the product or don't buy the product, i did some analysis of the dataset to show description of the dataset and gives some information of the dataset, i did some feature engineering to clean some data and select the features that good for the model.
