Exploratory Data Analysis

Exploratory Data Analysis or well known as EDA in this project is about analysis the dataset to show description of the dataset and gives some information of the dataset before doing modelling, show characteristic of each label in the dataset for easy to understand of the dataset before doing modelling and gives information before doing feature engineering in this dataset.


Feature Engineering

Feature Engineering in this dataset to make the dataset cleaned and make dataset good for modelling, in feature engineering in this dataset i removed the feature to prevent overfit in this dataset so the model easily classify the label, i transformed the feature to one hot encoding before doing modelling with this dataset, and prevent dummy trap in this dataset.


Modelling

Modelling in this dataset to classify the label for production, in the modelling i used k-fold and train test split in this dataset to compare of each model and finding best model for this dataset, for classify the label in this dataset i selected usefull features in this dataset and make classify.
